# Mytube App

This is Youtube Clone application

## Router

/ -> Home<br>
/join -> Login<br>
/login -> Login<br>
/search -> Search<br>
<br>
/users/edit -> Edit user<br>
/users/delete -> Delete user<br>
<br>
/videos/watch -> Watch video<br>
/videos/edit -> Edit video<br>
/videos/delete -> Delete video<br>
/videos/comments -> Comment on a video
